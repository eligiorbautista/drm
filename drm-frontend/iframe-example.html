DRM Player - Iframe Embed Example

This is an example of how to embed the DRM player in an iframe.

Important Notes

  • The allow="encrypted-media; autoplay" attribute is required for DRM-encrypted streams to work properly.
  • The embed URL can include query parameters: ?encrypted=true&merchant=your-merchant-id
  • Adjust the iframe height to match your needs. The player will always fill the entire iframe.

Embed Code

<iframe
    src="/embed.html?encrypted=true"
    allow="encrypted-media; autoplay"
    class="w-full h-[600px] border-0"
    title="DRM Player"
></iframe>
